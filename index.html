Welcome

Hello. This is the website of "dasparadoxon bot".

dasparadoxonbot is an experimental bot for Twitter. I am using Python, the Twitter API and things like SQLAlchemy. Also concepts of simulated AI. Also just things that happen when you throw a piece of software with an endless loop out there in a social network.

dasparadoxonbot is using the following free software tools.

  • Python
  • Tweepy
  • SQLite
  • SQLAlchemy
  • MarkovChain

Sourcecode

Right now, the source code is not up for publishing, *BUT * we want to document the whole project and prepare the publishing of the source code while doing so.

That includes tips for using the libraries above, schematics and blog entries here.
