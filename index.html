Welcome

Hello. This is the website of "dasparadoxon bot".

It is an experimental bot for Twitter. I am training here Python, using the Twitter API and things like SQLAlchemy. Also concepts of simulated AI. Also just things that happen when you throw a peace of software with a endless loop out there in a social network.

It is done with the following free software tools.

  • Python
  • Tweepy
  • SQLite
  • SQLAlchemy
  • PyMarkovChain

Hardware

It is running on a Raspberry 2 !

Sourcecode

Right now, the source code is not up for publishing, *BUT * we want to document the whole project and prepare the publishing of the source code while doing so.

That includes tips for using the libraries above, scematics and blog entries here.
